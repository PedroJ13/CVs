# Crear CVs

## Objetivo

Crear CVs ajustados al rol requerido, usando siempre como base los CVs reales existentes en `C:\Work\CVs\Base`.

## Fuente de verdad

Los CVs deben basarse unicamente en documentos reales dentro de:

- `C:\Work\CVs\Base\Pedro_Gutierrez_CV.docx`
- `C:\Work\CVs\Base\Pedro_Gutierrez_CV.pdf`

## Reglas

- No inventar puestos.
- No inventar experiencia laboral.
- No inventar empresas, fechas, titulos, certificaciones, tecnologias ni logros.
- Se permite reorganizar, resumir o destacar informacion real para ajustarla al rol requerido.
- Se permite adaptar el lenguaje al JD, siempre que el contenido siga siendo verdadero.
- Si el JD pide algo que no aparece en los CVs base, se debe indicar que no hay evidencia suficiente para incluirlo.
- Antes de crear un CV ajustado, revisar el JD y los documentos base.

## Proceso recomendado

1. Solicitar el JD si no fue enviado.
2. Leer los CVs base disponibles en `C:\Work\CVs\Base`.
3. Identificar requisitos, responsabilidades y palabras clave del rol.
4. Mapear cada punto del JD contra experiencia real del CV base.
5. Ajustar resumen, skills y bullets solo con informacion verificable.
6. Guardar el CV generado con un nombre que indique el rol o empresa.

## Criterio de ajuste

El CV final debe sentirse especifico para el rol, pero no debe agregar informacion que no exista en la experiencia real.
